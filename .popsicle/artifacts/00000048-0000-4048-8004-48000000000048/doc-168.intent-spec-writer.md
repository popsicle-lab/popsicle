## Block 4 — TelemetryReportFailOpen

```intent
type TelemetryReportResult {
  walReadable: Bool
  mainExitZero: Bool
}

intent TelemetryReportFailOpen(r: TelemetryReportResult) {
  require r.walReadable == false
  ensure r.mainExitZero'
  ensure r.mainExitZero' == true
}
```

语义：WAL 不可读时 report 仍 exit 0（fail-open）。
