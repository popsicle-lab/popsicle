# RFC: agent-runtime 执行面分层

> **Status**: Review

> **ADR**: ADR-001-agent-runtime-execution-layer.md

## Context

PROJ-81 需在不变更 skill-runtime 状态机前提下，实现手机派活 → 本机 Daemon → subprocess popsicle + Agent CLI。cli-ux 保持 IO shell（ADR-007）。

## Goals

- agent-server：队列、鉴权、WS、run 镜像 API
- agent-daemon：认领、执行、adapter、心跳
- cli-ux：`daemon` 子命令族
- 状态真相：本地 `.popsicle/state.db`

## Non-Goals

- Server 执行 AI；复活 sync；P0 Squad

## Proposed Design

见 `products/agent-runtime/ARCHITECTURE.md` 模块图与 File Manifest。

Daemon 执行循环：poll → claim → popsicle JSON CLI → prompt 组装 → cursor-agent → events。
