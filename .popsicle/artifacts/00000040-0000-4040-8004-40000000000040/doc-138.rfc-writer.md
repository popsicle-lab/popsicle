# RFC：feature-arch-spec pipeline

> PROJ-65。扩展 ADR-029，新增已有 product 大增量 full spec 链。

## Summary

新增 bundled pipeline **`feature-arch-spec`**：`prd → arch-debate → rfc → adr → intent-spec → intent-check → living-docs`。无 `facts`、无 `product-debate`。完成后接 **`feature-delivery`**。

## Motivation

| 场景 | 现有 pipeline | 缺口 |
|------|---------------|------|
| 小增量、无架构 ADR | `feature-spec` | 够用 |
| 新 product 绿地 | `product-greenfield-spec` | 过重（product-debate） |
| 迁移 slice | `migration-slice-spec` | 含 legacy facts |
| **已有 product 大增量 + ADR** | 无 | **本 RFC** |

## Stage 定义

| Stage | Skill | depends_on | requires_approval |
|-------|-------|------------|-------------------|
| prd | prd-writer | [] | true |
| arch-debate | arch-debate | [prd] | true |
| rfc | rfc-writer | [arch-debate] | true |
| adr | adr-writer | [rfc] | true |
| intent-spec | intent-spec-writer | [prd, adr] | false |
| intent-check | intent-consistency-check | [intent-spec] | false |
| living-docs | living-doc-author | [intent-check] | true |
